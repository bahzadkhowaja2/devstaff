# DevStaff Coding Task

#### Database Name
```devstaff```

#### Database Scripts
Run scripts from the below path

``` src/main/resources/database-scripts```

### APIs

To save Harvest
POST ```/api/harvested```

GET To get Harvest by id
```/api/harvested/{id}```

POST To save Plant
```/api/planted```

GET To get Plant by id
```/api/planted/{id}```

GET To get report by season
```/api/report/{season}```
