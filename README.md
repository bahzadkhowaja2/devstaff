# DevStaff Coding Task

#### Database Name
```devstaff```

#### Database Scripts
Run scripts from the below path

``` src/main/resources/database-scripts```

### APIs

To save Harvest
```POST /api/harvested```

To get Harvest by id
```GET /api/harvested/{id}```

To save Plant
```POST /api/planted```

To get Plant by id
```GET /api/planted/{id}```

To get report by season
```GET /api/report/{season}```
